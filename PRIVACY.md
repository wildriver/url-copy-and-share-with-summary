# Privacy Policy / プライバシーポリシー

## English

This Privacy Policy describes how the "URL Copy with Summary" Chrome extension handles your data.

### 1. Data Collection and Usage
The "URL Copy with Summary" extension **does not collect, store, or transmit any personally identifiable information** to any external servers owned by the developer.

* **Local Storage**: The extension stores your preferences, such as your selected AI provider (Groq or OpenRouter), API keys, and language settings locally on your device using the Chrome Storage API (`chrome.storage.sync`). This data is synchronized across your devices if you are signed into Chrome with sync enabled, but it is never accessed by the developer.
* **Active Tab Data**: To function, the extension accesses the URL, title, and text content of your active tab. This data is only processed temporarily when you click the extension icon or trigger its functions to generate summaries or eye-catch images.

### 2. Third-Party Services
If you choose to use the AI summarization or AI title generation features, the extension will transmit the text content of your active page to the AI provider you have configured:
* **Groq**: [Groq Privacy Policy](https://groq.com/privacy-policy/)
* **OpenRouter**: [OpenRouter Privacy Policy](https://openrouter.ai/privacy)

The data sent to these third-party APIs is governed by their respective privacy policies. The extension only sends the data necessary to fulfill your request (the page content and prompt) and uses your own API key.

### 3. Changes to This Policy
We may update this Privacy Policy from time to time. Any changes will be reflected in this document.

### 4. Contact
If you have any questions or concerns about this privacy policy, please contact the developer via the GitHub repository issues page.

## 日本語

本プライバシーポリシーは、Chrome拡張機能「URL Copy with Summary」におけるデータの取り扱いについて説明するものです。

### 1. データの収集と利用
「URL Copy with Summary」拡張機能は、ユーザーの**個人を特定できる情報を収集、保存、または開発者が所有する外部サーバーへ送信することは一切ありません**。

* **ローカル保存**: 拡張機能は、選択したAIプロバイダー（GroqまたはOpenRouter）、APIキー、言語設定などのユーザー設定を、Chrome Storage API (`chrome.storage.sync`) を使用して端末上にローカル保存します。Chromeの同期機能が有効な場合、このデータはユーザーのデバイス間で同期されますが、開発者がこのデータにアクセスすることはありません。
* **アクティブタブのデータ**: 拡張機能の機能を提供するため、現在アクティブなタブのURL、タイトル、およびテキストコンテンツにアクセスします。これらのデータは、拡張機能のアイコンをクリックしたり、要約やアイキャッチ画像の生成機能をトリガーした際にのみ一時的に処理されます。

### 2. サードパーティサービス
AIによる要約機能やAIタイトル生成機能を使用する場合、拡張機能はアクティブなページのテキストコンテンツを、ユーザーが設定した以下の対象AIプロバイダーへ送信します：
* **Groq**: [Groq プライバシーポリシー](https://groq.com/privacy-policy/)
* **OpenRouter**: [OpenRouter プライバシーポリシー](https://openrouter.ai/privacy)

これらのサードパーティAPIへ送信されるデータは、各プロバイダーのプライバシーポリシーに従って取り扱われます。拡張機能は、ユーザーのリクエスト（ページのコンテンツとプロンプト）を完了するために必要なデータのみを送信し、ユーザー自身のAPIキーを使用します。

### 3. 本ポリシーの変更
本プライバシーポリシーは随時更新される場合があります。変更があった場合は、本ドキュメントに反映されます。

### 4. お問い合わせ
本プライバシーポリシーに関するご質問や懸念事項がある場合は、GitHubリポジトリのIssueページより開発者へお問い合わせください。
