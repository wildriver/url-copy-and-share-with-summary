# URL Copy with Summary

様々なフォーマットでのURLコピー、AIによる要約生成、スマートなハッシュタグ抽出、そして魅力的なSNS用アイキャッチ画像を作成できる、インテリジェントなChrome拡張機能です。

## ✨ v2.3.x の新機能
- **アプリ名とロゴの刷新**: 新しい洗練された「URL Copy with Summary」ロゴと名称。
- **5種類のアイキャッチテンプレート**: 「モダングラデーション」「クラシックホワイト」「ネイビープロフェッショナル」「ダークポップ」「エレガントゴールド」から選択可能。
- **AIアシストアイキャッチ**: AIがアイキャッチ画像に最適な、短くてインパクトのあるタイトル（30文字以内）を自動生成します。
- **動的ハッシュタグ**: ページの内容に基づいて、コンテキストに沿ったハッシュタグをAIが自動抽出します。
- **スマートQRコード**: 長いURLに合わせてサイズを自動調整し、便利な「QRコードをダウンロード」ボタンを追加しました。
- **UIの自由度**: テキストエリアの縦方向リサイズが可能になり、より使いやすくクリーンなインターフェースに進化しました。

## 🚀 主な機能
- **プレミアムデザイン**: グラスモーフィズムを取り入れた、洗練されたダークモード対応のモダンUI。
- **バイリンガル対応**: 英語と日本語に完全対応。
- **URLコピー**: タイトル、要約、ハッシュタグを組み合わせてURLをクリップボードにコピー。
- **AI要約機能**: GroqまたはOpenRouterを使用してページの要約を生成（言語・文字数設定可）。
- **アイキャッチ画像生成**: 5種類のプレミアムテンプレートから、1200x630pxの美しいシェア用画像を即座に作成。
- **Amazon URL最適化**: 長いAmazonの商品URLを自動的に短くクリーンに整形。

## 📥 インストール方法
1. このリポジトリから最新の `extension.zip` をダウンロードします。
2. ダウンロードしたファイルを**解凍（展開）**します。
3. Chromeを開き、アドレスバーに `chrome://extensions/` と入力して移動します。
4. 画面右上の**「デベロッパーモード」**をオンにします。
5. **「パッケージ化されていない拡張機能を読み込む」**をクリックし、先ほど解凍した**フォルダ**（`manifest.json`が含まれているフォルダ）を選択します。

## ⚙️ 初期設定
1. 拡張機能のポップアップを開きます。
2. **設定（Settings）**をクリックし、使用するAIプロバイダー（GroqまたはOpenRouter）を選択し、APIキーを入力します。APIキーが必要な機能にはアスタリスク（`*`）が付いています。

## 🛠 開発情報
- **Manifest V3**: 最新のChrome拡張機能標準に準拠。
- **Canvas API**: プライバシーとパフォーマンスに配慮したローカルでの画像生成。
- **AI統合**: 任意のLLMプロバイダーと連携可能。
